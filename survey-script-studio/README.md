# Survey Script Studio

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 19.2.19.

## Prerequisites

Before running this application, ensure you have the following installed:

- **Node.js**: v22.14.0 or higher
- **Angular CLI**: v19.2.19
- **Angular**: v19.2.0

### Installing Angular CLI

If you don't have Angular CLI installed globally, run:

```bash
npm install -g @angular/cli@19.2.19
```

## Installation

1. Navigate to the project directory:
```bash
cd survey-script-studio
```

2. Install dependencies:
```bash
npm install
```

## Development server

To start a local development server, navigate to the project directory and run:

```bash
cd survey-script-studio
ng serve
```

Or use npm:
```bash
cd survey-script-studio
npm start
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.
